# DCGAN-chainer
DCGANのchainerによる実装。Trainerなんかを使って最近のバージョン向けに書いた。

## 動作環境
- [chainer](https://github.com/pfnet/chainer) >= 1.15.0
- [Pillow](https://github.com/python-pillow/Pillow)
- [matplotlib](https://github.com/matplotlib/matplotlib)
